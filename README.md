This project is a learning exercise in implementing the **Builder design pattern** using Java. The goal is to **understand and implement the Builder pattern** in Java, learn how to **structure a project using Gradle**, practice **modularizing code** with interfaces and concrete builders and demonstrate the **flexibility of the Builder pattern** for object creation.
